HISTORIA — WORLD 1

NARRADOR DEL REINOLa historia de Daniel...
INICIO
Mi trayectoria en el desarrollo de software comenzó en en la Universidad Tecnológica de Panamá en Coclé. Con una mentalidad analítica, construí bases sólidas en lógica de programación y algoritmos, utilizando lenguajes como C y Java para aprender a resolver problemas complejos a través del código.
BASES
La ingeniería me proporcionó una visión arquitectónica de los sistemas. Apliqué estos conceptos en proyectos que fusionaban el software con el mundo físico, desarrollando soluciones de redes e Internet de las Cosas (IoT) mediante el uso de beacons y tecnología Bluetooth.
FRONTEND
La necesidad de crear mejores experiencias me llevó a dominar la web. Consolidé mis bases estructurando interfaces con HTML, CSS y JavaScript puro, para luego evolucionar hacia el desarrollo de aplicaciones reactivas y escalables utilizando Angular, TypeScript y gestión de estados con NgRx.
BACKEND
Para dar vida a sistemas completos, me involucré en el lado del servidor construyendo APIs robustas con .NET, Clean Architecture y bases de datos SQL Server.A la par, llevé mis desarrollos a producción gestionando mi propia infraestructura con Docker, Nginx y CasaOS.
ACTUALIDAD
Tras consolidar mi experiencia profesional como pasante en LogicStudio implementando seguridad empresarial con JWT y Azure Entra ID, mi enfoque actual no se detiene. Hoy puedo construir soluciones automatizadas, integro entornos cloud en Azure y exploro la nueva generación de AI Agents con LiveKit.
